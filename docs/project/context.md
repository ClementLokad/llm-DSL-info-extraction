# Contexte & Objectifs du Projet

## Le Projet Scientifique Collectif (PSC)
Ce projet s'inscrit dans le cursus de l'**École Polytechnique (X24)**. Il vise à résoudre un problème scientifique complexe en collaboration avec un partenaire industriel.

## Collaboration avec Lokad
**Lokad** est leader dans l'optimisation de la supply chain via l'analyse quantitative. Leur plateforme repose sur le langage **Envision**. 
L'objectif du PSC est de faciliter la maintenance et l'évolution des scripts Envision massifs en fournissant aux développeurs un assistant capable d'expliquer la sémantique métier derrière le code.

## Objectifs Principaux
1.  **Réduire la dette technique** : En facilitant la compréhension des scripts legacy.
2.  **Améliorer l'onboarding** : Permettre aux nouveaux recrues d'être opérationnels plus rapidement.
3.  **Garantir la fiabilité** : Fournir des réponses basées sur des faits extraits directement du code source.
